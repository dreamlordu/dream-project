# 🌙 Tabir-i Rüya

Tabir-i Rüya, kullanıcıların rüyalarını bilimsel, geleneksel ve spiritüel perspektiflerden analiz eden gelişmiş bir yapay zeka destekli rüya yorumlama platformudur.

🔗 GitHub Repository: https://github.com/dreamlordu/dream-project

🔗 Live Demo: https://tabiri-ruya.vercel.app

---

# 📋 Proje Hakkında

Tabir-i Rüya, kullanıcıların gördüğü rüyaları detaylı şekilde analiz ederek anlamlandırmayı amaçlayan modern bir web uygulamasıdır.

Platform, Google Gemini AI teknolojisini kullanarak rüyaları üç farklı yorumlama kategorisinde analiz eder:

- 🧠 Psikolojik Yorumlama
- ☪️ İslami Yorumlama
- 🔮 Mistik Yorumlama

Modern kullanıcı arayüzü, animasyon destekli deneyimi ve yapay zeka altyapısıyla akıcı bir kullanım sunar.

---

# ✨ Özellikler

## 🔍 Çok Boyutlu Rüya Analizi
Rüyaları farklı ekollere göre yorumlama desteği.

## 🤖 Yapay Zeka Destekli Yorumlama
Google Gemini AI ile güçlü ve dinamik analiz sistemi.

## 💬 Gerçek Zamanlı Sohbet Deneyimi
Akıcı ve kullanıcı dostu chat arayüzü.

## 🎨 Modern UI/UX
Tailwind CSS ve DaisyUI destekli modern tasarım sistemi.

## ✨ Gelişmiş Animasyonlar
GSAP ve Framer Motion ile desteklenen geçiş efektleri.

## 📱 Responsive Tasarım
Mobil, tablet ve masaüstü cihazlarda tam uyumlu yapı.

## 🔐 Güvenli API Yönetimi
Environment variable sistemi ile güvenli API kullanımı.

# 📦 Kullanılan Paketler

```bash
npm install react react-dom
npm install vite
npm install tailwindcss daisyui
npm install framer-motion
npm install gsap
npm install @google/genai
```

# 💻 Kurulum

Projeyi yerel ortamda çalıştırmak için aşağıdaki adımları izleyin:

```bash
# Repoyu klonlayın
git clone https://github.com/dreamlordu/dream-project.git

# Proje klasörüne girin
cd dream-project

# Bağımlılıkları yükleyin
npm install
```

---

# 🔑 Environment Variables

Proje dizininde `.env` dosyası oluşturun:

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

---

# 🚀 Projeyi Çalıştırma

```bash
npm run dev
```

# 🌐 Deployment

Proje Vercel platformu üzerinde deploy edilmiştir.

Deploy işlemi için:

```bash
npm run build
```

Vercel üzerinde environment variable tanımlamayı unutmayın:

```env
VITE_GEMINI_API_KEY
```
